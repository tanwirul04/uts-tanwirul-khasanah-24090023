## Deskripsi singkat project (Fungsi Tiap Halaman)
Project ini merupakan website sederhana dengan 3 halaman utama: Login, Dashboard, dan Produk.  
Setiap halaman memiliki peran dan fungsi berbeda untuk mensimulasikan aplikasi.

### **1. Halaman Login (index.html)**
Pada halaman ini terdapat form login yang meminta email dan password.  
Sebelum masuk ke dashboard, input akan dicek terlebih dahulu untuk memastikan email dan password tidak kosong. Jika valid, pengguna akan diarahkan ke halaman dashboard.

**Fungsi:**
- Menerima input email dan password dari pengguna.
- Melakukan validasi sederhana (tidak boleh kosong).
- Mengarahkan pengguna ke Dashboard jika input valid.
- Menampilkan pesan jika form belum diisi.

Halaman ini berfungsi sebagai gerbang awal sebelum masuk ke sistem.

### **2. Halaman Dashboard (dashboard.html)**
Halaman dashboard berfungsi sebagai tampilan ringkas untuk melihat data secara keseluruhan.

**Fungsi:**
- Menampilkan rangkuman data dalam bentuk card:
  - Total Products
  - Total Sales
  - Total Revenue
- Ikon pada setiap card menggunakan Font Awesome.
- Menampilkan sidebar navigasi.
- Mengarahkan pengguna ke halaman list data produk melalui navigasi Lihat Data Produk.

### **3. Halaman Produk (products.html)**
Halaman ini berfungsi sebagai tempat pengelolaan data produk secara langsung.

**Fungsi:**
- Menampilkan daftar produk dalam tabel.
- Menyediakan fitur manipulasi data menggunakan JavaScript DOM:
  - **Edit produk**  
    Menampilkan form edit & memperbarui data langsung di tabel.
  - **Hapus produk**  
    Menghapus produk dari list dengan konfirmasi.
    
## Aspek yang Digunakan
- **HTML** – struktur dan konten halaman  
- **CSS** – styling, layout, Flexbox, card, table, & sidebar  
- **JavaScript (DOM, event)** – Validasi login, edit produk, hapus produk, manipulasi DOM, array dummy
- **Font Awesome** – ikon pada halaman login, dashboard dan halaman produk
